# Inventory Management API (Database PoC) 📦

A backend service demonstrating advanced database modeling and ORM usage for **Google Summer of Code 2026**.

## 🚀 Overview
This project showcases a robust schema for inventory management using **Drizzle ORM** and **PostgreSQL**. It includes relational data modeling, automatic timestamps, and unique constraints.

## 🛠️ Tech Stack
- **ORM:** Drizzle ORM
- **Database:** PostgreSQL
- **Language:** TypeScript
- **Migration Tool:** Drizzle-kit

## 📊 Database Schema
- **Products:** Tracks items, SKUs, stock levels, and categories.
- **Categories:** Organizes products into logical groups.

## ⚙️ Development
1. Install dependencies:
   ```bash
   npm install
   ```
2. Generate SQL migrations:
   ```bash
   npx drizzle-kit generate
   ```
